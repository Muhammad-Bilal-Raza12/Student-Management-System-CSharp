# Student-Management-System-CSharp
# 🎓 Student Management System (Mid-DB Project)
**University of Engineering and Technology (UET) Lahore** **Department of Computer Science**

## 📌 Project Overview
This is a desktop-based management tool developed to automate academic record-keeping. The system bridges the gap between a **C# (.NET WinForms)** graphical interface and a **MySQL** relational database. It focuses on the complex relationships between students, attendance, and multi-layered evaluations (Rubrics & CLOs).

## ✨ Core Features
* **Student Administration:** Full CRUD (Create, Read, Update, Delete) operations for student profiles.
* **Smart Attendance:** A dynamic interface allowing instructors to mark attendance for an entire class with a "Check-All" feature.
* **Advanced Evaluation:** A marks entry system that links **Assessment Components** to specific **Rubric Levels**, ensuring data integrity.
* **Database Logic:** Utilizes `UPSERT` (Insert or Update) logic to prevent duplicate entries while allowing record edits.
* **Real-time Search:** Optimized text-filtering for quick navigation through large student datasets.

## 🛠️ Technical Stack
* **Language:** C# (.NET Framework)
* **Database:** MySQL 8.0
* **IDE:** Visual Studio 2022
* **Architecture:** Relational Database Management System (RDBMS)
